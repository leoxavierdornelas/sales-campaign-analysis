Sales Campaign Performance Analysis (CRM)

Descrição
Projeto de análise de dados focado em eficiência de canais de aquisição (SMS, Voz, WhatsApp). O objetivo é calcular o ROI (Retorno sobre Investimento) e a Taxa de Conversão de leads em vendas.

Conexão com o Negócio
Este projeto simula a rotina de um Analista de Dados em empresas de Automação de Marketing e CRM (como o ecossistema TOTVS/Ligue Lead), onde é crucial entender qual canal traz o lead mais qualificado pelo menor custo.

Principais Insights Gerados

Comparativo de eficiência: SMS vs WhatsApp.

Análise de "Queima de Budget" com números inválidos.

Cálculo de ROI automatizado.

Tecnologias

Python (Pandas, NumPy)
